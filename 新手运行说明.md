## 适用人群

本说明面向**零基础新手**，不需要你改任何代码。你只需要：
- 安装 Python
- 安装依赖
- 双击运行脚本

---

## 第一步：用 Cursor 打开项目

- 打开 Cursor
- 点击 **File → Open Folder**
- 选择本项目文件夹 `ContestTrace`

---

## 第二步：检查 Python 是否已安装

在 Cursor 里打开一个终端（Terminal），输入：

```bash
python --version
```

你应该看到类似 `Python 3.11.x` 或 `Python 3.12.x`。  
本项目要求 **Python >= 3.10**。

---

## 第三步：安装依赖（只做一次）

在项目根目录（能看到 `requirements.txt` 的位置）执行：

```bash
pip install -r requirements.txt
```

如果提示 `pip` 找不到，换成：

```bash
python -m pip install -r requirements.txt
```

如果下载很慢（常见），可尝试：

```bash
python -m pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

---

## 第四步：一键运行（推荐）

### Windows（最简单）

直接双击 `run.bat`  
它会自动：
- 先做环境检查
- 再打开主菜单（爬取/查询/导出）

### Mac/Linux

在终端执行：

```bash
chmod +x run.sh
./run.sh
```

如果你不想用脚本，也可以直接运行：

```bash
python run_app.py
```

---

## 第五步：如何爬取数据

运行后会出现主菜单，输入：
- `1` 运行爬虫

爬虫会按配置的页码范围爬取，并显示：
- 当前页码
- 当前标题
- 是否入库成功/跳过（重复/筛选）
- 最终统计（新增/跳过/失败）

中途想停止：直接按 **Ctrl + C**，程序会**优雅中断**并保留已入库数据。

---

## 第六步：如何查询数据（不用写SQL）

主菜单输入：
- `2` 数据查询

你可以：
- 查看最近公告列表
- 按标题关键词搜索
- 按ID查看公告完整详情（含全文与结构化字段）
- 查看统计
- 更新处理状态（例如把 `new` 改成 `done`）

---

## 第七步：如何导出 Excel / CSV

主菜单输入：
- `3` 一键导出

可选择：
- Excel（`.xlsx`）
- CSV（`.csv`）

导出文件会自动保存到：
- `exports/` 目录

如果提示“文件被占用”，请先关闭正在打开的Excel再导出。

---

## 第八步：Web 可视化界面（Streamlit）

安装依赖后执行：

```bash
python run_web.py
```

浏览器会自动打开（默认端口以终端提示为准）。可在页面中检索竞赛、查看看板、维护个人画像、使用智能推荐（无 API Key 时为规则推荐）、导出数据。

---

## 第九步：GitHub Actions 自动爬取（可选）

将项目推送到 GitHub 后，工作流 `.github/workflows/crawl.yml` 可按计划运行爬虫并更新 `data/reports/latest_snapshot.csv`。  
详细步骤见 **`GHA_DEPLOY.md`**。本地可运行 `scripts/gha_setup.bat` 或 `scripts/gha_setup.sh` 模拟流程。

## 第十步：编码自测（可选）

```bash
python tests/test_encoding.py
```

用于验证 `encoding_fixer` 对 UTF-8 / GB18030 样本的解码是否正常。

## 常见问题（FAQ）

### 1）运行时报 `ModuleNotFoundError`

说明依赖没装好，回到“第三步”重新安装：

```bash
python -m pip install -r requirements.txt
```

### 2）运行时报网络错误/超时

这是正常的网络波动。项目已内置：
- 超时
- 重试
- 随机延时

你也可以在 `config/config.yaml` 里把：
- `timeout_seconds` 调大
- `max_retries` 调大

### 3）爬不到数据/解析失败

可能是官网DOM结构变化。解决办法：
- 先查看 `logs/` 目录当天日志，看看“未找到列表容器/正文容器”等提示
- 再打开 `static_pages/` 的原始HTML（如果开启保存）
- 最后按 `配置说明.md` 修改 `config/config.yaml` 里的 DOM 选择器
